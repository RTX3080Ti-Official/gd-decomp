Geometry Dash 2.2 Decompilation
===============================

Warning: The game CANNOT run without it's ressources obtained buy purchasing the game on Steam. (https://store.steampowered.com/app/322170/Geometry_Dash)

============
Introduction
============

This decompilation is made using Ghidra, which is a free decompiling tool.
This is an hobby project, you may contribute via adding functions/variables name

====
Goal
====

The current goal is to make a full version compilable with most compilers (GCC/Clang/MSVC)
I am not planning to make a byte matching or instruction matching decomp, the better is to get the game running.
